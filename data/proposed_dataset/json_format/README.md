# Json format of dataset
This dataset is intended for models like E2E/BiLSTM/BERT. The raw data has not been split with project domains.
The ranges of projects are shown below:
- Angular: 0~26
- Appium: 27~50
- Docker: 51~77
- Deeplearning4j: 78~103
- Ethereum: 104~128
- Typescript: 152~178
- NodeJS: 179~203
